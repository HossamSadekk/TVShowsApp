
# TVShows App

Used Component:-
- Java
- Room
- Retrofit
- Rxjava
- Picasso 
- ViewPager2
- RecyclerView

Features:-
- Showing the most popular Tv Series
- Search
- Add to Watchlist
- Delete from Watchlist
- Showing details for a specific TVShow
- Showing the Website & Episodes
